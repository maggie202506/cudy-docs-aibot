# Quick Installation Guide

## Scenario 1: Set up main unit for a New Mesh Network.

1. Plug the power adapter into the DC power jack of the main unit. Power it on and wait for the system LED to turn solid on.

2. Plug the Ethernet cable from the DSL/cable modem or wall outlet into Port 1 of the main unit.
    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/m1200_connection.webp" alt="" width="500px" style="display: block; margin: 0 auto;" />

3. Go to *Settings > WLAN* on your phone. Find and tap on the Wi-Fi name of the main unit. Enter its password and tap Join.
    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/join_wifi.webp" alt="" width="300px" style="display: block; margin: 0 auto;" />

    (Please find the Wi-Fi name and password on the product label at the bottom)
    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/find_ssid.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />    

4. Open a browser, enter [cudy.net](http://cudy.net) (or [192.168.10.1](http://192.168.10.1)) in the address bar, and create an administrator password to log in. Then follow the on-screen instructions to complete Quick Setup. Or simply launch the downloaded Cudy APP and follow the on-screen instructions.

    - Create administrator password to log in. 
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/wireless_router/login.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />
    
    - Select *Operation Mode* and click *Next*.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/qs2.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />
    
    - Select *Time Zone* and click *Next*.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/qs3.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />

    - Auto-detect or select *WAN Mode* (connection type), and click *Next*.
        - PPPoE: Enter the Username and Password provided by your lSP, if which is forgotten, please consult with your ISP.
            <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/qs4.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />
        - DHCP: Customize Hostname or keep it as default.
            <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/qs5.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />

    - Customize 2.4G and 5G SSID (wireless network name) and Password, or keep it default, and click *Next*.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/qs6.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />

    - Confirm the settings and click *Save & Apply*. Or click *Back* to modify it.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/qs7.webp" alt="" width="300px" style="border: 1px solid #eee; display: block; margin: 0 auto;" />

    !!! Note
        - It may take a few minutes for the router to successfully connect to the Internet.
        - lf you change the 2.4G/5G SSlD or Password, please connect to its new Wi-Fi network after the Quick Setup steps.

5. Place the new unit near the main unit, power it on and wait for its system LED to turn solid on. Choose one method to pair it with the main unit. Or simply follow the instructions on the downloaded Cudy APP.
    - Method A: Pair up automatically if the new unit is from the same package with the main unit.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/pair_a.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />
    - Method B: Press the WPS buttons on both units simultaneously.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/pair_b.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />
    - Method C: Plug one end of an Ethernet cable into the new unit, and the other end into the main unit.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/pair_c.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />

6. Move the newly added unit to a midpoint between the main unit and the Wi-Fi dead zone, where your phone detects at least two bars of signal. It will automatically reconnect to the main unit after booting.
    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/relocate.webp" alt="" width="500px" style="display: block; margin: 0 auto;" />


## Scenario 2: Add a new unit to an existing Mesh Network.

1. Place the new unit near the main unit, power it on and wait for its system LED to turn solid on. 

2. Choose one method to pair it with the main unit. Or simply follow the instructions on the downloaded Cudy APP.

    - Method A: Pair up automatically if the new unit is from the same package with the main unit.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/pair_a.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />
    - Method B: Press the WPS buttons on both units simultaneously.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/pair_b.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />
    - Method C: Plug one end of an Ethernet cable into the new unit, and the other end into the main unit.
        <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/pair_c.webp" alt="" width="400px" style="display: block; margin: 0 auto;" />

3. Move the newly added unit to a midpoint between the main unit and the Wi-Fi dead zone, where your phone detects at least two bars of signal. It will automatically reconnect to the main unit after booting.
    <img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/mesh_system/relocate.webp" alt="" width="500px" style="display: block; margin: 0 auto;" />
