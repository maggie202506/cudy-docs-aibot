# WISP
<img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/wireless_router/wisp.webp" alt="" width="800px" style="border:1px solid #eee; display:block; margin:0 auto;" />

On the **General Settings >> WISP** page, you can enable the router to wirelessly connect to a distant Wi-Fi network and share that connection locally via Ethernet/Wi-Fi, acting as a client bridge.  Please follow the steps below.

1. Turn on *Enabled* to activate the WISP function.
2. Determine 5GHz or 2.4GHz WiFi network you want to connect to. And then click *Scan* to search for available Wi-Fi networks. If necessary, click *Rescan* to re-search or update the host network list.
<img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/wireless_router/wisp4.webp" alt="" width="800px" style="border:1px solid #eee; display:block; margin:0 auto;" />
    - *BSSID*: Displays the MAC address of the host network.
    - *SSID*: Displays the visible name of the host Wi-Fi network.
    - *Channel*: Displays the frequency band the host network uses.
    - *Signal*: Displays the host network's signal strength.

3. Select your target host network and click *Connect* to pop up the connection page.

4. Enter the password if required and click *Save & Apply*.
<img src="https://cdn.jsdelivr.net/gh/Cudytech-pr/User-Guide/docs/images/wireless_router/wisp5.webp" alt="" width="500px" style="border:1px solid #eee; display:block; margin:0 auto;" />

**Success!**
